# IrrigationSystem

This project represents the IoT project that handles Plants irrigation optimization and plant caring.

It is created with Java 21, Spring Boot 3.4.5 for the back end and for front end [Angular CLI](https://github.com/angular/angular-cli) version 19.2.13.

## Start a back end server

Open the terminal in the root directory of the project and type

```bash

./gradlew bootRun

```

## Angular client server

To start a local development server, run:

```bash
npm install

ng serve
```
